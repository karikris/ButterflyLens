# ButterflyLens

## Australia’s Live Butterfly Evidence Map

**Discover, review, and strengthen Australia’s butterfly data.**

ButterflyLens is a community-verification and research-operations platform for comparing Atlas of Living Australia (ALA) baseline occurrence evidence with butterfly-image candidates discovered on Flickr.

Social-media search results are hypotheses—not biodiversity records. Machine screening finds candidate evidence; independent community and expert review determine how much that evidence can be trusted.

The project’s public discovery scope is:

> All butterfly candidate images discoverable through the published ButterflyLens Flickr search plan.

ButterflyLens never claims exhaustive Flickr coverage and never silently promotes a discovery candidate to an occurrence.
