## Subtask workflow

1. Inspect local source and tests.
2. Run required GitHits/Valyu or log `not-needed`/`unavailable`.
3. Implement one subtask.
4. Run targeted tests.
5. Run lint/type checks and `git diff --check`.
6. Update provenance.
7. Commit with required trailers.
